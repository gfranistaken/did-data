# Source PDFs

Drop raw DID PDF files here. These are the source documents used to produce the JSON files in `dids/`.

## Naming Convention

Name each file to match its DID document number, e.g.:

```
DI-SESS-81248B.pdf
DI-SESS-81223A.pdf
```

## Workflow

1. Drop PDF here
2. Convert to JSON using the DID schema (see `SCHEMA.md`)
3. Save the JSON to `dids/` with the matching slug filename
4. Commit and push both files together
